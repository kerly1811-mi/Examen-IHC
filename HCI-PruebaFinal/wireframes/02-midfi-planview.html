4. VALIDACIÓN EN TIEMPO REAL (Nielsen #4, #9)
MEJORA: Cambiar de validación onBlur (lenta) a onChange (tiempo real)
Flujo Actual (LENTO)
User escribe → onBlur → Validar → Guardar → UI actualiza
⏱ Retardo: usuario espera confirmación
Flujo Propuesto (RÁPIDO)
User escribe → onChange → Validar INLINE → UI feedback
↓ (después 1s) → Auto-guardar
✓ Feedback inmediato + Auto-save
Estados de Campo
✓ VÁLIDO: Border verde + ícono check + "Guardado hace X"
⚠ REQUERIDO: Border amarillo + ícono warning + "Complete este campo"
✗ ERROR: Border rojo + ícono error + mensaje específico
○ NEUTRAL: Border gris + sin ícono (sin validación aún)
